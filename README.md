# Vehicle Data Visualization Web Application

## Project Description
This project is a web application built with Streamlit that allows users to explore and visualize vehicle sales data. Users can filter the dataset by vehicle model and model year, then interact with several visualizations that provide insights into vehicle characteristics and pricing.
The main goal of this project was to practice building interactive web applications, creating data visualizations, and deploying applications to the cloud.

## Features
* Filter vehicle data by model and model year
* Interactive data table
* Vehicle type distribution by model
* Vehicle condition distribution by model year
* Price distribution by manufacturer/model
* Average vehicle price by vehicle type

## Technologies Used
* Python
* Pandas
* Streamlit
* Plotly Express
* Render

## Dataset
The application uses a dataset containing vehicle listings, including information such as:
* Model
* Model year
* Price
* Odometer
* Vehicle type
* Condition

## How to Run Locally

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app_sprint7_vehicles.py
```

## Deployment
The application is deployed using Render.
